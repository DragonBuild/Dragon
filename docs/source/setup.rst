Setup
---------------------

Installing 
*********************

Installing is incredibly simple::

    pip3 install dragon 

Type "dragon" in your terminal to complete the initial setup 

Updating
*********************

Versions 1.6.0 and later::
    
    dragon update

Updating from earlier versions::
    
    rm -rf ~/.dragon 
    pip3 install --force-reinstall dragon 
    dragon
